# AI Plantation

This project is a multi-layered AI system for generating and validating novel, creative, and scientific content.

## Components

- **Motherboard**: The dynamic source of truth.
- **Approver GOD**: The validation and promotion pipeline.
- **Baby AIs**: Specialized, domain-expert AIs that generate content based on approved knowledge.

See `src/app.py` for an example of how to run the system.
